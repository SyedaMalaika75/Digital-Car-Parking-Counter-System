# 🚗 Digital Car Parking Counter System

## 📌 Overview

The Digital Car Parking Counter System is a Digital Logic Design project developed to automatically count the number of vehicles entering and exiting a parking area.

The system uses Digital Logic ICs, JK Flip-Flops, IR Sensors, and Seven-Segment Displays to monitor and display the available parking count in real time.

---

## 🎯 Project Objectives

- Count vehicles entering the parking area.
- Count vehicles exiting the parking area.
- Display the total number of parked vehicles.
- Demonstrate Digital Logic Design concepts using hardware components.

---

## 🛠 Components Used

- Breadboard
- JK Flip-Flops
- Digital Logic ICs
- IR Sensors
- Seven-Segment Display
- Resistors
- Push Buttons
- Connecting Wires
- 5V Power Supply

---

## ⚙ Features

- Automatic vehicle counting
- Entry and Exit detection
- Real-time display
- Hardware implementation
- Digital Logic based design
